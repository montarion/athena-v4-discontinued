# athena-ui
## Vue webfront for Athena V4

## Project setup
run in the athena-ui folder within any CLI:
```
npm install
```

### Compiles and hot-reloads for development
run in the athena-ui folder within any CLI:
```
npm run serve
```

### Compiles and minifies for production
run in the athena-ui folder within any CLI:
```
npm run build
```

### Lints and fixes files
run in the athena-ui folder within any CLI:
```
npm run lint
```
